# PRD — Telegram-бот для закрытого клуба

## 1. Общее описание

Проект представляет собой Telegram-бота для управления платным доступом в закрытую группу.

Цель: протестировать гипотезу платного сообщества с минимальными затратами и быстрым запуском.

---

## 2. Бизнес-цели

- Проверить спрос на платный доступ к экспертному контенту
- Протестировать воронку привлечения через амбассадоров
- Обеспечить автоматизацию:
  - приема оплат
  - управления доступом
  - уведомлений

---

## 3. Целевая аудитория

- Профессионалы отрасли
- Пользователи Telegram
- Клиенты, пришедшие через рекомендации (амбассадоры)

---

## 4. Основные сценарии использования

### 4.1 Новый пользователь
- приходит в бота
- регистрируется
- вводит промокод (опционально)
- оплачивает подписку
- получает доступ в группу

---

### 4.2 Активный пользователь
- состоит в группе
- получает напоминания о продлении
- продлевает подписку

---

### 4.3 Истекший пользователь
- удаляется из группы
- получает уведомление
- может оплатить снова

---

## 5. Функциональные требования (верхний уровень)

- регистрация пользователей
- прием оплат
- управление подписками
- выдача доступа в Telegram-группу
- удаление по окончании подписки
- напоминания о продлении
- система промокодов

---

## 6. Ограничения

- MVP (минимальная версия)
- без сложной админки
- без сложной реферальной системы
- без аналитики (на первом этапе)

---

## 7. Метрики успеха

- количество оплат
- конверсия в оплату
- удержание пользователей
- эффективность промокодов
